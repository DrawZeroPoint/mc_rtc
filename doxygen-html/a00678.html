Detailed Description

This file holds all mc_rtc::Configuration based serialization operation for mc_rbdyn objects.

This file contains classes that are used to configure the visual styles of various elements in the GUI.

This header contains the base block of all elements in the GUI system.

This file contains some utility template metaprogramming functions for the GUI.

For the implementation details of specific elements look into their dedicated header
